Основные принципы организации управления доступом:
  • Для доступа к определенным разделам программы каждому Пользователю назначаются одна или несколько ролей
  • Каждому Пользователю при регистрации автоматически назначается роль 'Новый сотрудник'. С этой ролью пользователь может просматривать и редактировать свой профиль.
  • Роли можно группировать в каталоги. Так же возможно перемещать роли из одного каталога в другой, для этого необходимо просто захватить роль левой кнопкой мыши и перетащить в необходимый каталог
  • Каждая роль может иметь один из трех статусов: АКТИВНЫЙ, 'Роль (Каталог) удалена', 'Действие роли (каталога ролей) приостановлено'
  • Каждая роль является контейнером для набора прав
  • Роль не может быть с пустым набором прав
  • При изменении набора прав Роли у конкретных пользователей которым назначена эта Роль изменения наступят после обновления токена авторизации
